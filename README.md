# 📚 POO - Classes

Repositório dedicado aos estudos e exercícios de **Programação Orientada a Objetos (POO)** com **Java**.

## 📖 Conteúdo

Este repositório contém aulas e exercícios práticos sobre conceitos fundamentais de POO, incluindo:

- **Classes e Objetos** - Fundamentos e instanciação
- **Encapsulamento** - Modificadores de acesso e getters/setters
- **Herança** - Extensão de classes e reutilização de código
- **Polimorfismo** - Sobrescrita de métodos e interfaces
- **Abstração** - Classes abstratas e interfaces
- **Composição** - Relacionamento entre objetos
- **Estruturas de Dados** - Listas, pilhas, filas, etc.

## 🎯 Objetivo

Consolidar os conhecimentos adquiridos em aula através de exercícios práticos e implementações de conceitos de POO.
